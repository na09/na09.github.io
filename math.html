Arithmetic Properties

The Commutative Properties

  • The Commutative Property of Addition
    • x + y = y + x

    i.e 2 + 5 = 5 + 2
  • The Commutative Property of Multiplicative
    • x * y = y * x

    2 * 5 = 5 * 2

The Associative Properties

  • The Associative Property of Addition
    • a + (b + c) = (a + b) + c 

    2 + (3 + 4) = (2 + 3) + 4
  • The Associative Property of Multiplicative
    • a * (b * c) = (a * b) * c 

    2 * (9x * 3) = (2 * 9x)* 3

The Distributive Properties

This arithemtic property allows us to foil factors. 

 a * (b + c) = a * b + a * c 

 2 * ( 3 + 5 ) = 2 * 3 + 2 * 5

The Identity Properties

  • The Associative Property of Addition:

    • The sum of any number with zero equals to the original number

    x + 0 = x 

    5 + 0 = 5
  • The Associative Property of Multiplicative

    • The product of any number with 1 equals to the original number

    x * 1 = x 

    5 * 1 = 5

The Inverse Properties

  • The Associative Property of Addition

    • Any number adds to the orignal number so that the sum equals 0 (zero).

    x + (-x) = 0

    2 + (-2) = 0
  • The Associative Property of Multiplicative

    • Any number multiplies to the original number so that the product equals to 1 (one). 

    x * (1/x) = 1

Slope of a Linear Line

Slope of a linear line is rise over run. It can be calculated by using the following formula:

$$m = {y_2-y_1\over x_2-x_1} ,$$

where m is the slope of the line and $$(x_1, y_1)$$ and $$(x_2, y_2)$$ are points on the line.

Or using the alternative formula,

$$(y - y_1) = m (x - x_1)$$

Equation of the Line and Graph of Linear Line

To write an equation, we need a slope of the line and one point on the line:

$$y= {mx + b}, $$

where and are slope and y-intercept of the line.

For example,
The equation with a slope of 4 and pass through a point (1,2) is $$2 = {4(1) + b}$$ $$b = -2 $$ Then, $$y= 4x -2 $$

Factoring with Quadratic Equation

The standard form of quadratic formula is: 
ax2 + bx + c = 0 or ax2 + Sx + P =0 

Where S and P are Sum and Produt of the roots.
  • Factor Using the X- method

    i.e. x2 + 4x + 4 To use this method, make sure your quadratic equation is in standard form. To factor this binomial we need to find two x values that add up to 4 (S = 4) and multiply together to get P = 4.The first step is to list all the multiples of 4
  • Factor by Group Factoring

    i.e. x2 + 5x + 6 = 0
    x2 + 2x + 3x + 6 = 0
    (x2 + 2x ) + (3x + 6 ) = 0
    x( x + 2) + 3( x + 2) = 0
    (x + 2)(x + 3)
  • Factor Using Quadratic Formula:
    $$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
    i.e. x2 + 6x + 9 = 0
    where a = 1, b = 6, c = 9 then, $$x = {-6 \pm \sqrt{6^2-4(1)(9)} \over 2(1)}.$$ x = -3, +3 Then,
    (x - 3)(x + 3)
